## 💻 개발환경 <br>
React + TypeScript + Vite <br> <br>


## 🔎 주요 기능 <br>
- 실시간 울음 분석 - 배고픔, 불편함, 트름, 아픔, 피곤함으로 분류 <br>
- 실시간 pose 추정을 통한 질식사 및 낙상 방지 <br>
- 일일 통계 분석 제공 <br> <br>
